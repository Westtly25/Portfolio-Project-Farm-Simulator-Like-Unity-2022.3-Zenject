# A Farming simulator game on Unity 2022.3
## Description
### This project is a clone of popular game Stardew valley where player has own farm, can plant plants, harvest and e.t.c
## Technology stack : 
  1. Zenject,
  2. UniRx,
  3. DoTween,
  4. .Net c#,
  5. Unity 2022,
  6. Unity Addressables,
  7. Unity UI Toolkit,
  8. Unity Localization
